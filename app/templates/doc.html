API de Dados V1

Esta pagina descreve a API de dados, uma forma automatizada de obter os dados do COVID-19. Em geral de uso de desenvolvedores de softwares ou estudantes que desejam integrar essas informações a outros sistemas. A documentação abaixo descreve, com exemplos, como utilizar a interface de dados

Acesso à API de todos os casos

O acesso à API é publico, não é necessário criar uma conta tampouco autenticar

Estrutura das requisições

URL

É necessário fazer uma chamada HTTP, método GET, para a URL
https://covid-19.lucas-martins.com/api/v1/all_cases
